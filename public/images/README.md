## How to Download

1. Open each URL in your browser
2. Right-click on the image and select "Save Image As..."
3. Save with the exact filename mentioned above
4. Place all images in this `/public/images/` directory

## Alternative: Use Placeholder Images

If you want to use your own images temporarily:
- Hero background: Any high-quality landscape image (minimum 1920x1080px)
- Product images: Square or 4:3 ratio images (minimum 800x600px)
